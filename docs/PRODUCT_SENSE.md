# MealMate Lite — 产品感知

## 一句话定位

以 AI 对话为核心交互的家庭饮食规划 Android App。

## 目标用户

私人家庭（1-6 人），负责"今天吃什么"的那个人。不想每天想菜、不想忘记买菜、不想浪费食材。

## 核心价值

解决"每天吃什么"这个高频低价值决策。对话式交互，不填表单、不学操作，说一句话就行。

## 与通用 ChatGPT 的区别

| ChatGPT | MealMate Lite |
|---------|---------------|
| 不记得你家有什么菜 | 持久化菜品库 |
| 不知道冰箱里有什么 | 追踪库存（v0.3+） |
| 每次对话从头来 | 多轮上下文 + 历史偏好 |
| 纯文字无法操作 | 可交互确认面板 |
| 无提醒能力 | 提醒推送（v0.8+） |
| 无法多端共享 | 家庭多设备同步 |

## v0.1 范围边界

**做**：
- AI 对话管理菜谱（增删改查、批量生成）
- AI 对话生成/调整周计划
- 设备认证（bootstrap + 家庭码）
- 多设备同步
- 4 页 App（对话、菜品、计划、设置）

**不做**：
- 库存、采购清单、提醒、统计
- 一餐多菜、角色偏好
- CV/语音、Web 端、iOS

## 交互原则

- **对话优先**：对话页是首页和主入口
- **渐进式**：菜品只需一个名字就能存在，细节后续慢慢补
- **低门槛**：30 秒建立首批库存，一句话批量生成菜品库

## 业务统一语言

| 术语 | 含义 |
|------|------|
| Recipe | 菜品/菜谱 |
| WeeklyPlan | 一周的餐次安排 |
| PlanItem | 哪天哪餐吃什么 |
| Conversation | 设备级 AI 对话 |
| DeviceToken | 设备认证令牌 |
| FamilyCode | 家庭邀请码 |
| PendingConfirmation | 批量操作的服务端预览草稿 |
